Setup Steps · Git Bash on Windows
1
Prerequisite
Check Python is installed
Open Git Bash and verify Python works. You need Python 3.7 or higher.
$ python --version Python 3.11.x # If that fails, try: $ python3 --version
⚠
If Python is not installed, download it from python.org/downloads — check "Add Python to PATH" during install, then restart Git Bash.
2
Navigate
Go to the folder with your files
Use the Git Bash path format — drive letter becomes /c/ instead of C:\
# Example — change to match where you saved the files: $ cd /c/Users/YourName/Downloads/eigenfaces # Verify all files are there: $ ls generate_eigenfaces.py eigenfaces_template.html eigenfaces_sklearn.html
3
Install
Install Python packages
Only needs to be done once. Pillow is optional — the script has a fallback if it's missing.
$ pip install scikit-learn numpy pillow
# If pip is not found: $ python -m pip install scikit-learn numpy pillow
4
Run
Run the Python script
This downloads the real Olivetti dataset (~5 MB from figshare.com), runs sklearn PCA, and generates the HTML. Takes about 15–30 seconds total.
$ python generate_eigenfaces.py
# Expected output: ============================================================== Eigenfaces: Real sklearn PCA + Olivetti Faces → HTML ============================================================== [1/5] Fetching Olivetti faces dataset via sklearn... X.shape = (400, 4096) [2/5] Running PCA(n_components=50).fit(X)... [3/5] Downsampling 64×64 → 32×32... [4/5] Encoding to base64... [5/5] Building eigenfaces_olivetti.html... ============================================================== ✓ eigenfaces_olivetti.html (2100 KB) ==============================================================
✓
Success! eigenfaces_olivetti.html will appear in the same folder. This file has the real 400 Olivetti faces embedded — no internet needed to open it.
5
Open
Open the result in your browser
The generated HTML is fully self-contained — no server, no dependencies, just open it.
$ start eigenfaces_olivetti.html
# Or double-click the file in Windows Explorer # Or drag it into Chrome / Firefox
Troubleshooting

Common Errors on Windows Git Bash

⚠
'python' is not recognized Try python3 instead of python, or reinstall Python and check "Add to PATH".
⚠
'pip' is not recognized Run python -m pip install scikit-learn numpy pillow instead.
✗
Download error / URLError / figshare blocked Your network is blocking figshare.com. Try a different network, mobile hotspot, or VPN. The script only needs internet for step [1/5] — the download is ~5 MB.
⚠
eigenfaces_template.html not found The script must be in the same folder as the template. Run ls to confirm both files are present.
⚠
Pillow not found / ImportError Pillow is optional. The script will use a numpy fallback automatically. Just re-run — it will still work.
ℹ
Want to preview before running anything? Open eigenfaces_sklearn.html right now — it works with simulated data and shows the full interface.

Original vs reconstructed pairs using pca.inverse_transform(). Adjust k slider to see quality change.

Noisy query → nearest neighbor in sklearn eigenspace → top-8 L2 distances shown live.
